内容简介

——为什么我天天都在用火腿棒？——

——为什么很多人离了警钟就感到不适？——

——所以，是时候让原版那贫瘠的武器选择来点新花样了——

武器改动
1.三尾猫鞭

-- 造成连续四段的17伤害攻击，这个多段伤害不受角色倍率影响

-- 使用缝纫道具修复（如针线包，胶带）

-- 耐久为0时不会消失，仍然可以攻击但失去一切特殊效果（类似警钟）

-- 鞭挞的触发概率提高至30%（对BOSS仍是5%）

2.蝙蝠棒

-- 攻击时若玩家处于满血状态，蝙蝠棒的伤害将提高至51并且恢复1点耐久

-- 若不处于满血状态，伤害降低为34并消耗1耐久为玩家恢复生命，玩家生命越低恢复越多（相对的，越接近满血越难恢复，最低只恢复1.5血量）

-- 耐久为0不消失，失去特殊效果变为17伤害，直到玩家下次满血挥刀时恢复耐久

-- 移除攻击会消耗精神值的设定

-- 蝙蝠（包括UM的大蝙蝠）现在有1%的概率直接掉落蝙蝠棒

基于重做后的特性，蝙蝠棒现在仅有40点耐久并且无法对发条生物，食人花，哈奇切斯特触发特效 此外，蝙蝠棒现在将一击秒杀所有种类的蝙蝠并使它们有35%额外掉落一个蝙蝠翅膀

3.刺耳三叉戟

-- 对目标造成6秒的潮湿效果，非常适合搭配辣闪

-- 当处于潮湿状态时缓慢恢复耐久(这下成祖传用品了)

-- 配方中的树枝替换为活木，所需独角鲸角的数量降低为1

4.铥矿棒

-- 召唤的暗影小触手史诗级优化，在召唤的瞬间就能够打出两次平A（基本上是完全不会丢伤害了）且现在也能正常对影怪造成伤害

-- 小触手的召唤概率提高至30%，如果同时穿戴了铥甲铥头，那么概率将提高至70% （套装效果）

-- 增速效果重做，移除10%常驻加速，取而代之的是现在铥棒将在战斗状态下提供25%加速效果！

-- 耐久提高至325，攻击距离变为1

-- 配方调整为2活木4铥矿8燃料

5.玻璃刀

-- 玻璃刀的耐用程度现在随着使用者的理智升高而提高

-- 具体计算机制为-取当前san值百分比（向下取整）/100等到当前耐久损耗，san值低于25%时取0.25，也就是最高4倍耐用

-- 削弱暗影生物，使其降低50%移速和30%伤害

-- 击杀暗影生物额外奖励12理智

6.栅栏击剑

-- 攻击距离提高至1,攻击自动取消后摇，强化手感

7.晨星

-- 伤害提高至34，持续时间(耐久)提高至1.5天

-- 可用土豆，硝石，闪电羽毛恢复一定耐久

-- 放置在两种发电机旁边时自动消耗发电机电量恢复耐久

-- 薇诺娜使用时将对机械生物 （所有发条生物, wx78 ,双子魔眼）造成额外42,5伤害

注：即便开启永不妥协晨星也会应用所有本MOD的改动，薇诺娜使用时对发条生物额外42.5伤害增强为对所有生物额外34伤害

8.暗影剑

-- 影刀的耐用程度现在随着使用者的理智降低而提高

-- 具体计算机制为-取当前san值百分比（向下取整）/100等到当前耐久损耗，san值低于20%时取0.2，也就是最高5倍耐用

9.暗影槌

-- 1级以上时物品名称下可看见饥饿度，并可通过喂食肉食恢复饥饿与耐久；可用作锤子；满级时获得攻速提高效果

10.战斗圆盾

-- 耐久度提高至1053

11.奔雷矛

-- 冲刺技能大幅优化，移除卡手前摇，增加滑步设定提高灵活性

12.拾荒尖帽

-- 耐久度提高至1260，可防洞穴落石，可用自动修理机修复

13.视界扩展器

-- 可防风。移除耐久度设定但本体更加脆弱，可以用锤子砸碎获得一些材料（但这也意味着其他具备破坏能力的生物也会破坏放在地面的视界扩展器）

14.天气风向标

-- 现在具有20次耐久且耐久耗尽不会消失，可以用鹅毛修复2点耐久，召唤出的龙卷风伤害降低到2.5点，但是可以足足持续60秒而非5秒（站场能力大幅度提高，显著提高对群怪的控制能力）


护具品改动
1.草甲

-- 现在提供+10%移速

2. W.A.R.B.I.S套装

* 获取时期提前到拾荒疯猪阶段

* 优化锁定追踪效果

* W.A.R.B.I.S头戴齿轮的减伤降低到70%，但增伤倍率改为1.15/1.2/1.25/1.3

* W.A.R.B.I.S盔甲的耐久度提高为1050(原730)

同时穿戴W.A.R.B.I.S头戴齿轮与W.A.R.B.I.S盔甲并且处于极致状态时，周期性的释放蒸汽冲击波击退锁定目标周围的敌人(吓我一跳释放忍术)


日用品改动
1.海花环

-- 负面理智效果提高至-12san/min，真正的月岛花环

2.懒人手杖

-- 耐久提高到30次，损坏时返还步行手杖

3.珍宝袋(新增)

-- 背包：科技要求-完整远古遗迹科技，制作配方-啜食者皮*6 + 铥矿碎片*2 + 橙宝石*1

-- 可存储所有Prefab字符串后缀为gem的物品 (编者注：兼容按规范为新增宝石命名的mod)

4.糖果袋(重做)

-- 取消装备机制改为便携容器机制，并新增打开贴图和格子贴图

-- 这个容器拥有50%的腐烂倍率

-- 可放入物品除了原版的万圣节食物与玩具，原版DST所有玩具以外，还新增所有冬季盛宴物品/零食和部分糖类料理（太妃，糖豆，棱镜的烤棉花糖，UM的米花糖，樱花林的樱花太妃）

-- 配方常驻

5.清洁扫把(重做)

-- 无科技需求5理智即可制造（随时随地，想换皮肤就换皮肤，不再受限！），卸下时消失

6.提灯

-- 允许玩家快速丢出提灯，并在丢出的瞬间切换物品栏内伤害最高的武器

-- 编者注：这个功能作者有单独mod版本

7.睡袋

-- 配方更改为蘑菇皮*1 + 中毒金丝雀*1 + 犁地草膏*2，只能做一个，丢出后不消失可重复使用，CD90s

-- 处于睡袋孢子云中的玩家会持续消耗饥饿恢复血量与精神

-- 处于睡袋孢子云中的玩家的随从会持续恢复大量生命

-- 处于睡袋孢子云中的 毒菌蟾蜍/苦难蟾蜍 将持续受到高额的百分比伤害 （对于不能分解刷皮的补偿，同时跟香炉一样也是一种以Boss装备对策Boss的思路）


杂项

-- 奔雷冲刺无前摇 （白令经典改了介绍不吱声）

-- 手持提灯时按V可开关投掷功能

-- 所有配置项现在都有具体的改动说明
